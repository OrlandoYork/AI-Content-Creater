# AI全媒体内容生产与分发系统

## 项目背景

传统内容生产存在效率低、风格不统一、分发渠道分散、效果难以量化等问题。本项目模拟新媒体运营团队的内容生产与分发流程，实现选题策划、内容创作、多平台分发与效果分析的全流程AI辅助。
## 核心业务流程

```
热点分析 → 选题策划 → 内容创作 → 内容审核 → 多平台分发 → 数据监测 → 效果分析 → 内容优化
```

## 核心功能模块

### 1. 热点分析与选题策划

- 热点采集：模拟全网采集各大平台的热点话题、热搜榜单、热门内容
- 热点分析：AI分析热点的热度、趋势、受众群体、情感倾向
- 选题生成：根据热点与账号定位，自动生成多个选题建议
- 选题管理：支持对选题进行筛选、修改、收藏与排期

### 2. AI内容创作

- 多类型内容生成：支持生成文章、短视频脚本、海报文案、社交媒体帖子等多种类型的内容
- 风格定制：支持定制内容风格（正式、幽默、文艺、专业等），保持账号风格统一
- 内容改写：对已有内容进行改写、润色与扩写，生成不同版本的内容
- 标题生成：根据内容生成多个吸引人的标题，供用户选择

### 3. 内容审核与多平台分发

- 内容审核：AI自动审核内容是否包含敏感信息、违规内容
- 多平台适配：根据不同平台的规则与特点，自动调整内容格式与发布时间
- 一键分发：模拟一键分发到微信公众号、微博、抖音、小红书等多个平台
- 发布管理：管理所有平台的发布内容与发布时间

### 4. 数据监测与效果分析

- 数据采集：模拟采集各平台的内容数据（阅读量、播放量、点赞量、评论量、转发量、粉丝增
长）
- 数据分析：分析内容的传播效果、用户画像、互动情况
- 效果评估：评估不同选题、不同内容类型、不同发布时间的效果
- 优化建议：根据数据分析结果，生成内容优化建议与选题方向

### 附加功能

- 素材库管理：管理图片、视频、音频等内容素材，支持标签化检索
- 评论管理：自动回复常见评论，过滤恶意评论
- 竞品分析：分析竞品账号的内容策略与运营效果

### 开发约束

- 不接入真实社交媒体平台API，所有数据采用模拟生成
- 内容生成基于扣子平台的文本生成能力实现
- 必须实现至少3种类型内容的创作与分发功能
- 要求生成可视化的内容效果分析报表
